# ShopSmart AI

An AI-powered E-Commerce Analytics & Recommendation System built with Streamlit that combines business intelligence, customer analytics, product performance monitoring, and personalized product recommendations in a single interactive platform.

The application provides:

* Interactive executive dashboards for sales and revenue analysis
* AI-driven product recommendation engine
* Product catalog exploration with advanced filtering
* Customer segmentation and churn-risk analysis
* Geographic sales visualization and regional performance tracking
* Product analytics including discount impact and return analysis
* Real-time shopping cart and checkout simulation

## Key Features

### Dashboard Analytics

* Revenue and order tracking
* Conversion and rating metrics
* Category-wise performance analysis
* Monthly sales trend visualization

### Product Intelligence

* Search and filter products across multiple categories
* Product performance monitoring
* Discount and pricing analysis
* Inventory visibility

### AI Recommendations

* Personalized recommendations based on user preferences
* Budget-aware product suggestions
* Category-based recommendation engine
* Recommendation session tracking

### Customer Insights

* Customer segmentation
* Lifetime value analysis
* Churn-risk identification
* Customer feedback management

### Geographic Analytics

* Interactive sales distribution maps
* Regional revenue comparison
* Geographic performance monitoring

## Tech Stack

* Python
* Streamlit
* Pandas
* NumPy
* Plotly

## Running the Application

```bash
pip install streamlit
streamlit run test_app.py
```

## Project Goal

ShopSmart AI demonstrates how data analytics, recommendation systems, and business intelligence dashboards can be integrated into a modern e-commerce platform to support data-driven decision-making and improve customer experience.
